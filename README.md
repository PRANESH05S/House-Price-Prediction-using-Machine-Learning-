House Price Prediction Model

This project builds a machine learning regression model to predict house prices using housing data.

Dataset

House Price dataset containing multiple features related to houses such as:

area

number of bedrooms

number of bathrooms

number of floors

other property features

Dataset Source:
https://www.kaggle.com/datasets/bhanupratapbiswas/house-price-prediction

Algorithms Used

Linear Regression

Random Forest Regressor

Gradient Boosting Regressor

Evaluation Metrics

The models were evaluated using:

RMSE (Root Mean Squared Error)

MAE (Mean Absolute Error)

Residual Analysis

Best Model

Gradient Boosting Regressor (Lowest RMSE)
